§ Функціонал

Що вже реалізовано

Від реєстрації до платформного адміністрування — повний фінансовий цикл у єдиній PWA з нативним рівнем досвіду.

01 💳
Кредити
Ануїтетний розрахунок, часткові погашення, трекер залишку та прогрес-бар на дашборді.
02 🏦
Депозити
Фіксована та плаваюча ставка, автоматичне нарахування відсотків, перегляд активних вкладів.
03 💱
Конвертер валют
Live курси Національного банку України, обмін між власними рахунками.
04 🛡️
Face ID / Passkeys
WebAuthn автентифікація, PIN-lock з idle-таймаутом, Bearer-сесія з авторотацією токена.
05 📈
Фінансовий рейтинг
Скоринг 0–100 на основі норми заощаджень, балансу та активності транзакцій.
06 📋
Бюджетний планувальник
Ліміти за категоріями, автокатегоризація транзакцій, прогрес-бари витрат на місяць.
07 📊
Аналітика
Витрати за типами, heatmap активності, спарклайн балансу, прогноз, top-отримувачі.
08 💸
Перекази та виплати
Переказ за номером рахунку або карткою, шаблони, ідемпотентність, виписка CSV/PDF.
09 🔔
Push-сповіщення
VAPID Web Push, фонові сповіщення, управління підписками, адмін-розсилка.
§ Модулі системи

Чотири контури
одної платформи

ARM Bank — це не лише API + мобільний вигляд, а зчеплені контури з чіткими межами відповідальності.

Відкрити PWA ↗
01
Банківська PWA (soldier)
React 18 + TypeScript + Vite. 17+ екранів: дашборд, кредити, депозити, FX, бюджет, аналітика, цілі, картки, профіль. Offline-режим, Face ID.
02
Flask REST API
Python 3, blueprints: auth, account, credit, deposit, feature, admin, operator, platform, push, analytics, agent, passkey. PostgreSQL/SQLite.
03
ARM Admin SPA
Окремий SPA: 14+ розділів — користувачі, картки, фінанси, звіти, AI-агент задач, документообіг з ECDSA-підписом.
04
Operator Panel
Вузький контур для нарахування виплат клієнтам з ідемпотентністю. Ролева ізоляція.
