Часті питання

Portfolio piece — повноцінна full-stack банківська платформа, написана з нуля. Демонструє навички роботи з Flask REST API, PostgreSQL, JWT-аутентифікацією, PWA та Android-розробкою на Kotlin.

Базовий URL: https://army-bank.onrender.com/api. Авторизація через Bearer JWT токен. Kotlin Retrofit інтерфейс та data classes доступні у репозиторії. Детальна документація на GitHub.

Kotlin Android клієнт — в розробці. SDK (Retrofit + data classes) вже написаний і підключається до REST API. APK буде доступний у GitHub Releases.

Viacheslav Munister — full-stack розробник. Backend на Python/Flask, фронт на HTML/CSS/JS + PWA, мобільний клієнт на Kotlin. Контакти: GitHub.
