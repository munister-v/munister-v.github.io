Часті питання

Як відкрити рахунок?

Зареєструйтесь на платформі, підтвердіть email, заповніть дані. Рахунок буде активний за хвилини. Це повністю онлайн процес.

Скільки коштує переведення коштів?

Переводи між рахунками повністю безкоштовні. Немає жодних комісій, неважливо який розмір суми.

Мої дані захищені?

Так, абсолютно. Всі дані шифруються, паролі зберігаються з використанням bcrypt, база даних захищена. Ми дотримуємося найвищих стандартів безпеки.

Чи може я отримувати пуш-сповіщення?

Так, усі операції супроводжуються миттєвими сповіщеннями. Ви будете знати про кожну операцію в реал-тимі.

Хто розробив ArmyBank?

ArmyBank розроблена Viacheslav Munister як сучасна фінансова платформа для військових. Весь код написаний з урахуванням найвищих стандартів якості та безпеки.

Чи є API для розробників?

Так! У нас є REST API з документацією, Kotlin data classes і Retrofit interface для розробників. Детальну інформацію читайте в документації.
