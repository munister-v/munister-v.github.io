FAQ

Частые вопросы

Почему авторизация на этом лендинге не “внутри страницы”?

Сайт размещен на GitHub Pages (статический хостинг). API и база работают в приложении банка на Render. Поэтому кнопка “Войти” ведет на /bank/ (и дальше — в банк).

Где админ‑панель?

Внутри приложения: откройте /bank/ → затем /admin (доступ по роли admin).

Можно ли сделать так, чтобы банк открывался прямо под доменом без редиректа?

Да, но нужен прокси (Cloudflare Workers/Pages Functions, Netlify/Vercel rewrites или VPS с Nginx). GitHub Pages сам по себе проксировать не умеет.
